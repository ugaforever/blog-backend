# Backend приложения-блога с использованием Spring Framework

Проект создан в рамках обучения на курсе **Мидл Java-разработчик** от **practicum.yandex.ru**.

## Задание
Напишите бэкенд приложения-блога на Java с использованием Spring Framework, работающий в сервлет-контейнере Tomcat/Jetty, выберите БД для хранения постов и комментариев и реализуйте интеграцию бэкенда с ней.

![Архитектура приложения-блога](https://pictures.s3.yandex.net/resources/image_1756926710.png)

1. Фронтенд работает на веб-сервере Nginx, принимает запросы от браузера пользователя на http://localhost:80/, отправляет запросы на получение данных в бэкенд по REST.
2. Бэкенд работает в сервлет-контейнере Tomcat/Jetty, принимает REST-запросы от фронтенда на http://localhost:8080/, отправляет SQL-запросы на создание/получение/изменение/удаление данных в БД PostgreSQL/H2 или любой другой, формирует ответ и возвращает фронтенду.
3. База данных хранит данные, обрабатывает запросы от бэкенда и возвращает результат.

## Чек-лист требований по условию задания
- [x] Spring Framework (версия 6.1 и выше), выполняющегося в любом из современных сервлет-контейнеров Tomcat/Jetty.
- [x] Не использовать проекте Spring Boot.
- [x] Код приложения хранится в Git-репозитории (на Github).
- [x] Сборка приложения осуществляется с помощью систем сборки Maven/Gradle. Выбор - Maven.
- [x] Код приложения написан на Java 21.
- [x] База данных приложения может быть как персистентной (например, PostgreSQL), так и в памяти (например, H2). Выбор - H2.
- [ ] Бэкенд получает запросы от фронтенда по REST, обрабатывает их, формирует ответ и возвращает его фронтенду.
- [ ] Бэкенд предоставляет следующие эндпоинты, которые использует фронтенд:
    - [x] Получение списка постов
    - [ ] Получение списка постов c поиском 
    - [x] Получение поста
    - [ ] Добавление поста
    - [ ] Редактирование поста
    - [x] Удаление поста
    - [ ] Инкремент числа лайков поста
    - [ ] Обновление картинки поста
    - [x] Получение картинки поста
    - [x] Получение всех комментариев поста
    - [x] Получение комментария поста
    - [ ] Добавление комментария к посту
    - [ ] Редактирование комментария к посту
    - [ ] Удаление комментария
- [x] Бэкенд запущен на http://localhost:8080.
- [ ] Бэкенд покрыт тестами (юнит- и интеграционными) с использованием JUnit 5, TestContext Framework и кеширования контекстов.

##  Сборка
```

sudo -u tomcat cp ./target/my-blog-back-app.war /opt/tomcat/webapps/ROOT.war
```
